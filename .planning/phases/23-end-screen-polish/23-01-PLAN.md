<objective>
Implement a simple "You Win" screen that shows after defeating Niggel (boss 2, the last boss in v1.0).

Current flow: Boss defeated → PostBossDialogue → AvanzarFase → `if (currentBossIndex < 4)` → Shop (but boss 3 doesn't exist).

New flow: Boss defeated → PostBossDialogue → AvanzarFase → `if (currentBossIndex >= 2)` → Show VictoryUI.

The VictoryUI is a simple full-screen overlay with:
- A placeholder Image (space for a sprite the user will add later)
- A congratulations text ("¡Ganaste!" or similar)
- "Menú Principal" button → VolverAlMenu()
- "Salir" button → Application.Quit()
</objective>

  <name>Task 1: Create VictoryUI script and modify AvanzarFase to trigger it</name>
  <files>
    Assets/Scripts/UI/VictoryUI.cs [NEW]
    Assets/Scripts/GameFlowManager/GameFlowManager.cs
  </files>
  <action>
1. **Create `VictoryUI.cs`** following the same pattern as `DeathUI.cs`:
   - A MonoBehaviour with serialized fields for:
     - `Image victoryImage` (placeholder for sprite)
     - `TextMeshProUGUI victoryText` (congrats message)
     - `Button botonMenu` (Main Menu)
     - `Button botonSalir` (Exit)
     - `CanvasGroup canvasGroup` or `GameObject panel` (to show/hide)
   - `Show()`: activate panel, pause game (Time.timeScale = 0)
   - `Hide()`: deactivate panel, unpause
   - `OnClickMenu()`: call `GameFlowManager.Instance.VolverAlMenu()`
   - `OnClickSalir()`: call `Application.Quit()` (and `#if UNITY_EDITOR UnityEditor.EditorApplication.isPlaying = false`)

2. **Modify `AvanzarFase()`** in `GameFlowManager.cs`:
   Change the `PostBossDialogue` case from:
   ```csharp
   if (currentBossIndex < 4)
   {
       currentBossIndex++;
       ChangeState(GameState.Shop);
   }
   else
   {
       VolverAlMenu();
   }
   ```
   To:
   ```csharp
   if (currentBossIndex >= 2)
   {
       // Final boss defeated — show victory screen
       Debug.Log("[GameFlowManager] Final boss defeated! Showing victory screen.");
       var victoryUI = FindObjectOfType<VictoryUI>(true);
       if (victoryUI != null)
       {
           victoryUI.Show();
       }
       else
       {
           Debug.LogWarning("[GameFlowManager] No VictoryUI found! Falling back to menu.");
           VolverAlMenu();
       }
   }
   else
   {
       currentBossIndex++;
       ChangeState(GameState.Shop);
   }
   ```

3. **Add `victoryUI` reference to `UIManager.cs`** if it uses explicit references, or ensure the VictoryUI GameObject is a child of the UIManager canvas so `FindObjectOfType<VictoryUI>(true)` can find it.
  </action>
  <verify>Unity compiles with zero errors. AvanzarFase shows VictoryUI when currentBossIndex >= 2.</verify>
  <done>VictoryUI.cs created. AvanzarFase triggers it after boss 2. UIManager has the reference.</done>
</task>
