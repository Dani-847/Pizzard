# Plan 7.1: Scene Assembly & Integration (Unity MCP)

## Objective
Use the Unity MCP to set up the playable scene flow, implementing the scripts across multiple scenes (one per boss), preserving player progress via the GameFlowManager, and using colored primitives as placeholders for assets.

## Context
- The user compiled successfully but the game flow is not hooked up in the Unity Editor.
- Need: `MainMenu`, `IntroDialog`, `Shop`, `PreBossDialog`, and 4 `BossArena` scenes.
- Need: Use circle primitives for Player and Bosses, reusing the structure of prototype walls.

## Proposed Changes

### GameFlowManager Integration
#### [MODIFY] GameFlowManager.cs
Add `currentBossIndex` ranging 1-4.
Modify `GetSceneForState(GameState.BossFight)` to `BossArena_` + `currentBossIndex`.
Modify `AvanzarFase` logic so switching from BossFight to Shop increments `currentBossIndex` (if < 4) and handles victory conditions.

### Scene Construction
*CRITICAL UNITY REQUIREMENTS:* Every scene must have a `Main Camera`. Every scene with UI components must have a `Canvas` and an `EventSystem`. All assembled scenes MUST be added to the Build Settings.

#### [NEW] MainMenu.unity, IntroDialog.unity, Shop.unity, PreBossDialog.unity
Standardize menus with a `Main Camera`, `Canvas`, `EventSystem`, buttons, and attach their respective `*UI.cs` components. 
`GameFlowManager` is initiated in `MainMenu.unity` and carries `DontDestroyOnLoad`. `ProgressionManager` will automatically attach to it.

#### [NEW] BossArena_1.unity, BossArena_2.unity, BossArena_3.unity, BossArena_4.unity
Set up a `Main Camera`.
Set up common prototype arena: Red/White boxes for walls (reused).
Set up Blue Circle for `PlayerController`, `PlayerHPController`, etc. 
Set up Green Circle for Boss `PblobController` (Arena 1) and similar for the other 3 Bosses.
Include `DeathUI` canvas and an `EventSystem` in each scene.

#### Build Settings Integration
Once scenes are created, they must be added to the `EditorBuildSettings` in this exact order:
0. Assets/FlowScenes/MainMenu.unity
1. Assets/FlowScenes/IntroDialog.unity
2. Assets/FlowScenes/Shop.unity
3. Assets/FlowScenes/PreBossDialog.unity
4. Assets/FlowScenes/BossArena_1.unity
5. Assets/FlowScenes/BossArena_2.unity
6. Assets/FlowScenes/BossArena_3.unity
7. Assets/FlowScenes/BossArena_4.unity
