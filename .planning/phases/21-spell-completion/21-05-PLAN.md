**PinaQuesoQuesoAttack.cs** (piña|queso|queso — becomes cone on impact):

  - Get travel direction: rb.velocity.normalized (or store the last move direction in a field updated each FixedUpdate)
  - Use Physics2D.OverlapCircleAll at transform.position with GameBalance.Spells.PinaQuesoQueso.ConeRadius
  - For each collider in the overlap:
    - Compute toTarget = (collider.transform.position - transform.position).normalized
    - Compute angle = Vector2.Angle(toTarget, travelDir)
    - If angle <= GameBalance.Spells.PinaQuesoQueso.ConeHalfAngle:
      - Deal GameBalance.Spells.PinaQuesoQueso.ConeDamage via BossBase.TakeDamage (PblobController fallback)
  - Then Destroy(gameObject)
- Track travel direction: in FixedUpdate (or Update), cache `lastVelocity = rb.velocity.normalized`
- All values from GameBalance.Spells.PinaQuesoQueso.*
- The absorption behavior comes free from the base class
