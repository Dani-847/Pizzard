# Plan 3.2: Wand & Element Combat Framework

## Objective
Implement the core wand casting system allowing the player to queue elements (Queso, Pepperoni, Piña) based on Wand Tier (1, 2, or 3) and evaluate the final spell cast.

## Context
- .gsd/SPEC.md
- Assets/Scripts/Player/WandController.cs
- Assets/Scripts/Elements/ElementTypes.cs
