# Plan 18.2: P'blob Phase 2 (Circle Minigame)

## Objective
Implement Phase 2 of the P'blob boss fight: the hidden color circle minigame.

## Mechanics
At 66% HP:
1. Boss returns to center, becomes invulnerable.
2. 30s timer UI appears above boss/below healthbar.
3. 3 circles spawn (2 Red, 1 Green, 2x player diameter). They are White by default.
4. Circles move randomly for 5s, then stop.
5. If player steps on a circle, it reveals its true color (Red/Green). Stepping off turns it White again.
6. Boss is ONLY vulnerable while the player stands actively inside the Green circle.
7. Phase ends when boss HP hits 33% OR timer hits 0 (restarting Phase 2).

## Tasks
1. **Circle Prefab:**
   - Create `PblobCircle` prefab (SpriteRenderer, CircleCollider2D as Trigger).
   - Add `PblobCircleController` script:
     - `enum CircleType { Red, Green }`
     - On `Start()`: Color = White.
     - `OnTriggerEnter2D`: If tag is "Player", change color to Red/Green. Notify Boss (if Green, boss becomes vulnerable).
     - `OnTriggerExit2D`: If tag is "Player", change color to White. Notify Boss (boss becomes invulnerable).
     - Provide `MoveRandomly(duration, speed)` coroutine.

2. **Phase 2 Logic (`PblobController.cs`):**
   - `Phase2Transition`: Move boss to center using Lerp. Clean up any remaining projectiles.
   - `Phase2`: 
     - Instantiate 3 circles (assign 2 Red, 1 Green randomly).
     - Command circles to move randomly for 5s (`GameBalance.Bosses.Pblob.Phase2CircleMoveTime`).
     - Spawn and start a 30s UI Timer floating above the boss.
     - While Phase2 is active, Boss vulnerability is strictly driven by the `PblobCircleController`s.
     - If timer hits 0 before 33% HP: Destroy circles, restart `Phase2` state.
   - Check HP Threshold: If HP falls to 33% (1/3 of MaxHP), destroy circles, toggle invulnerable, trigger `ChangeState(Phase3Transition)`.

## Verification

- Verify boss takes 0 damage if standing on Red/White. Verify damage taken when on Green.
- Verify timing out the 30s timer restarts the minigame.
- Verify dropping boss to 33% cleans up circles and advances state.
