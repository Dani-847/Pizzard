Use Unity MCP tools to find and wire LocalizedText components to static UI text GameObjects.

- Play button text → key: `menu_play`
- Options button text → key: `menu_options`
- Exit button text → key: `menu_exit`
- Continue button text → key: `menu_continue`

**Options Panel (OptionsUI):**
- Language label → key: `options_language`
- Volume label → key: `options_volume`
- Combinations label → key: `options_combinations`
- Accept button text → key: `options_accept`

**Combinations Panel (CombinationsUI):**
- Back button text → key: `options_back`

**Death Screen (DeathUI):**
- Retry button text → key: `death_retry`
- Exit to Menu button text → key: `death_exit_menu`
- Death title text (if exists) → key: `death_title`

**Approach:**
1. Use `find_gameobjects` (by_component: "TMP_Text" or "Text") to locate all text objects
2. For each identified static text object, use `manage_components` to add the `LocalizedText` component
3. Use `manage_components` set_property to set the `key` field on each LocalizedText

If some text objects are children of button GameObjects, search by name patterns like "Play", "Options", "Exit", etc.
