# 🍕 PIZZARD

PIZZARD es un proyecto desarrollado en **Unity** que experimenta con mecánicas de juego, balance de daño y sistemas básicos de gameplay.  
El objetivo del proyecto es crear una base modular donde se puedan probar y ajustar diferentes mecánicas de combate y comportamiento.

## 🚀 Características

- Sistema de combate básico
- Balance de daño configurable
- Scripts organizados por módulos
- Fácil de extender para nuevas mecánicas

## ⚙️ Balance del juego

Los valores de daño, ajustes y balance de mecánicas se encuentran en el siguiente archivo:

👉 https://github.com/Dani-847/Pizzard/blob/main/Assets/Scripts/Core/GameBalance.cs
